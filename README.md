# Pathfinder Game Project
A Python Pathfinder game developed collaboratively with my team members, showcasing maze navigation and basic pathfinding algorithms.
## Overview
A **Pathfinder game** developed in **Python** where the player navigates a maze or map to reach the goal while avoiding obstacles.  
The project demonstrates **basic game logic, pathfinding algorithms, and interactive programming**.

---

## Features
- Maze navigation  
- Player movement using keyboard input  
- Pathfinding algorithm (e.g., BFS, DFS, or A*)  
- Obstacles and goal detection  
- Simple console-based interface

---

## Technologies
- **Language:** Python  
- **Libraries:** Tkinter (for GUI) / Pygame (if used)  
- **Logic:** Pathfinding algorithms, loops, and conditional statements
